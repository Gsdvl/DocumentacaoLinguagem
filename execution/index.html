    G2DL - Interpretador de Linguagem

    Compilação

    Para compilar o projeto, utilize o comando:

    make

    Esse comando executa os seguintes passos automaticamente:

    • Gera o analisador léxico com Flex
    • Gera o analisador sintático com Bison
    • Compila todos os arquivos .c
    • Organiza os .o em build/
    • Cria o executável final em bin/g2dl_interpreter

    Execução

    Após a compilação, o executável estará disponível em:

    bin/g2dl_interpreter

    Para executar um programa da linguagem G2DL, use:

    ./bin/g2dl_interpreter arquivo.g2dl

    Certifique-se de que o main.c está preparado para ler o arquivo de entrada.

    Organização

    ├── Makefile
├── src/
│   ├── main.c
│   ├── ast/
│   ├── lexer/
│   ├── parser/
│   ├── interpreter/
│   └── utils/
├── build/         # Criado automaticamente
├── bin/           # Criado automaticamente
└── include/       # Headers, se usados

    Comandos Make Disponíveis

    Você pode usar os seguintes comandos com make:

    Comando O que faz
    make ou make all Compila todo o projeto e gera bin/g2dl_interpreter
    make clean Remove todos os arquivos gerados, incluindo build/, bin/ e temporários
    make debug Não implementado (mas pode ser usado para recompilar com -g)
    make help Exibe instruções de uso dos comandos do Makefile

    Exemplo de uso do clean:

    make clean

    Limpa todos os arquivos de build e recompila:

    make clean
make

    Exemplo de Script .g2dl

    a = 10;
b = 5;
soma = a + b;
printf("Resultado: %d\\n", soma);

mensagem = "Olá, " + "G2DL!";
printf("%s\\n", mensagem);

    Requisitos

    Certifique-se de ter as ferramentas necessárias:

    sudo apt install build-essential flex bison
